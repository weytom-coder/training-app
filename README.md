# Thomas Training PWA (GitHub Pages)

## Deploy (Web UI)
1. Neues Repository auf GitHub anlegen (z. B. `training-app`).
2. Inhalt **dieses Ordners** hochladen (Dateien liegen im Root).
3. Repository → **Settings** → **Pages** → Build and deployment:
   - **Source: Deploy from a branch**
   - **Branch: main** / **root**
4. Warten (1–2 Min) → URL: `https://<USER>.github.io/<REPO>/`.
5. iPhone: URL öffnen → **Teilen → Zum Home‑Bildschirm**.

## Update
- Dateien ersetzen/hochladen → GitHub Pages deployt neu.
- Falls alte Version geladen wird: Safari Website‑Daten löschen oder 1–2 Min warten (Service Worker Cache).
